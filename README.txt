An Erlang beanstalkd client.

Quick start:

  $ make
  ...
  $ erl -pa ebin
  ...
  1> {ok, Q} = beanstalk:connect().
  ...
  2> {inserted, ID} = beanstalk:put(Q, <<"hello world">>).
  ...
  3> {reserved, ID, <<"hello world">>} = beanstalk:reserve(Q).
  ...
  4> {deleted} = beanstalk:delete(Q, ID).
  ...
  5> ok = beanstalk:close(Q).
  ...
